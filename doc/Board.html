Overview

The Board class is used when you want to play TicTacToe for a game with one or more round.

Constant Summary

COURT_VALUE =

Assume value for board: 0 indicates the board court currently initial state, 1 for Player 1, and 2 for Player 2. From these value, we will calculate the game state. The win condition: sum of one of the row or column or diagonal column should be 3 when then player 1 won, and 12 when the player 2 won.

court value for second player must be greater than 3 times of first player court value.

def move(row, col)
  row, col = row.to_i, col.to_i
  # row and column is out of the grid cell
  return :out_of_range if !(0..2).include?(row) && !(0..2).include?(col)
  # Already value set for specific cell
  return :illegal_move if court[row][col] != COURT_VALUE[0]
  # all turn has been finish
  return :game_over if turn_complete?

  update_court(row, col)
  update_status
  switch_player
  return :next_move
end
